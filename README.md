## 📂 Project Highlights

### [SOC_Analyst](./SOC_Analyst/)
- Splunk SPL query library for phishing, brute force, lateral movement
- Realistic ticket simulations with triage decisions and escalation logic
- Alert severity classification matrix

### [Threat_Intelligence](./Threat_Intelligence/)
- Python IOC tracker with VirusTotal & AbuseIPDB integration stubs
- MITRE ATT&CK technique mapping for common attack patterns
- APT threat report template based on real CTI workflows

### [Vulnerability_Management](./Vulnerability_Management/)
- CVSS-weighted patch prioritization script (exploit availability, asset criticality)
- Nessus XML report parser — outputs CSV with risk ratings
- CVE triage decision tree aligned with SSVC framework

### [Incident_Response](./Incident_Response/)
- Full ransomware IR playbook (detection → containment → eradication → recovery)
- Breach response timeline template (GDPR/HIPAA notification windows)
- IR checklist for first responders

### [Cloud_Security](./Cloud_Security/)
- AWS IAM credential audit — finds over-permissioned users, stale keys
- S3 bucket hardening checker — public ACLs, encryption, logging gaps
- CloudTrail log analyzer — detects suspicious API calls & privilege escalation
